# Environment

#### Install Django

#### MongoDB Get and Run
```
sudo apt-get install mongodb
sudo service mongodb start
```

#### Python packages

```
sudo pip install -r requirements.txt
```

#### Run the application
To the run the application only one step needs to be taken.
```
cd stock_data
chmod u+x main.py
./main.py
```
